# Event ID 4953: Windows Firewall ignored a rule because it could not be parsed.

## Description
This event generates if Windows Firewall was not able to parse Windows Firewall rule for some reason.

## Data Dictionary
|Standard Name|Field Name|Type|Description|Sample Value|
|---|---|---|---|---|
|profile|Profile|UnicodeString|the name of the profile of the ignored rule.|`All`|
|reason_for_rejection|ReasonForRejection|UnicodeString|the reason, why the rule was ignored.|`An error occurred.`|
|rule_id|RuleId|UnicodeString|the unique identifier for ignored firewall rule.|`{08CBB349-D158-46BE-81E1-2ABC59BDD523}`|
|rule_name|RuleName|UnicodeString|the name of the rule which was ignored.|`-`|
